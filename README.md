# 2025-InnovateStateHackathon-MSU
This is a repository for the 2025 Innovate State Hackathon @ MSU and i'm uploading it as it might be useful as a somewhat blueprint for anyone else doing this in the future!

------ Libraries Used ------

ESP32Servo Library by Kevin Harrington, and John K. Bennett

Adafruit VusIO by Adafruit

Adafruit PWM Servo Driver Library by Adafruit

Fork of PS3 Contoller Host by Jeffery van Pernis (PLEASE make sure to use the fork one as it has the updated header file)
